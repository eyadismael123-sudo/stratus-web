Always write scripts in this exact format:

```

[Ready to copy-paste Instagram/TikTok caption]

[From the hashtag sets in marketing/hashtag-sets.md]
```

## Hook Rules (Non-Negotiable)

The hook is 80% of the video. If the hook is weak, nothing else matters.

### Hook Formulas That Work:
1. **Pattern interrupt:** Start mid-sentence or with something unexpected
2. **Bold claim:** "I replaced a $3,000 agency with a Telegram bot"
3. **Mystery:** "I've been building something for 3 weeks and I can't show you yet"
4. **Relatability:** "Things only [specific group] understand"
5. **Controversy:** "Everyone is using AI wrong. Here's why."
6. **Numbers:** "$50/month. That's what it costs to never write a LinkedIn post again."

### Hook Anti-Patterns (Never Do):
- "Hey guys, so today I wanted to talk about..." (instant swipe)
- Starting with a logo or intro animation (nobody cares)
- "In this video I'm going to show you..." (just show it)
- Any greeting or pleasantry before the hook
- Throat clearing, "um", "so basically" (cut these in edit)

## Pacing Rules

- **One idea per 5 seconds.** If a 30-sec video has 2 ideas, it's too slow.
- **Cut every 3-4 seconds.** Jump cuts keep attention. Static shots lose it.
- **Text on screen matches audio.** People watch on mute. Captions are mandatory.
- **End 2 seconds after the last useful thing.** Don't linger. Don't add "thanks for watching."
